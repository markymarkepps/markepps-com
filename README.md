# markepps.com — Personal blog

This repository contains the source for a personal blog built with Hugo and the PaperMod theme.

The site lives in this repository as a standard Hugo project. The theme used is `PaperMod`, located in the `themes/PaperMod` folder. The generated public output is in the `public/` directory (this folder is usually generated by Hugo and may be ignored in deployments that build from source).

## Quick start

- Requires Hugo (extended recommended).
- Run locally: `hugo server -D` and open http://localhost:1313/.
- Build for production: `hugo` → output in `public/`.

## Layout

- Theme: `themes/PaperMod`
- Content: `content/`
- Static files: `static/`

## Deployment

Build locally and publish `public/`, or let your hosting (Netlify, Vercel, Cloudflare Pages) build the site with `hugo`.

## Notes

- Use `hugo new` to create posts. Set `draft: false` for published posts.
- For SCSS/assets you need the Hugo extended binary.
